============================================
  电脑出厂调试工具 — 使用说明书
  龙信硬件组
============================================

本工具用于仓库发货前对新电脑（Windows 10/11）进行一键式系统设置、
软件安装、Windows 激活和网络驱动器映射。

============================================
【一、准备工作】
============================================

在运行本工具之前，请按以下目录结构准备好文件：

  脚本目录/
  ├── 运行_GUI工具.bat         ← 推荐！双击启动图形界面
  ├── main_gui.ps1             ← 统一 GUI 程序
  ├── system_setup.ps1         ← 系统设置脚本
  ├── full_setup.ps1           ← 完整安装脚本（设置+软件）
  ├── system_restore.ps1       ← 还原脚本
  │
  ├── wallpaper.jpg / .png     ← 壁纸（可选，支持 jpg/png/bmp）
  ├── lockscreen.jpg / .png    ← 锁屏（可选，支持 jpg/png/bmp）
  │
  └── software/                ← ★ 所有安装程序放这里
      ├── *.exe / *.msi        ← 普通软件安装包
      ├── *火绒* / *sysdiag*   ← 火绒安全安装包
      ├── A-HEU_KMS_Activator  ← Windows 激活工具（可选）
      ├── jihuo.zip            ← 激活工具压缩包备份（可选）
      ├── usb_driver/          ← USB 驱动文件夹（含 android_winusb.inf）
      └── 诺咪雅MCR3512.../    ← MCR3512 控制键（含 setup.exe + .reg）

============================================
【二、快速启动】
============================================

1. 右键 "运行_GUI工具.bat" → 以管理员身份运行（推荐）
   或直接双击（脚本会自动申请管理员权限）

2. 在弹出的 GUI 窗口中点击对应按钮即可

============================================
【四、各功能详细说明】
============================================

4.1  仅系统设置（约 2-3 分钟）
──────────────────────────────────
执行以下操作，不安装软件：

  步骤1 — 设置壁纸
    自动查找脚本目录下的 wallpaper.jpg/png/bmp 并设为桌面壁纸

  步骤2 — 设置锁屏
    自动查找 lockscreen.jpg/png/bmp 并复制到系统目录，写入注册表策略

  步骤3 — 关闭 Windows 更新
    禁用 wuauserv / UsoSvc / WaaSMedicSvc / BITS 四个服务
    写入注册表策略禁止自动更新和自动重启
    禁用相关计划任务

  步骤4 — 电源管理
    显示器/睡眠/休眠全部设为永不息屏
    弹出确认窗口：
      选"是" → 台式机模式（电源按钮→关机，关闭快速启动）
      选"否" → 笔记本模式（仅合盖不操作）
    注意：中途弹出的"是/否"选择窗口可能在任务栏闪烁，请切换窗口确认

  步骤5 — 关闭 Windows 安全防护
    关闭篡改防护 → 防火墙 → Defender 实时/行为/脚本/入侵防护
    → SmartScreen → 信誉检查 → PUA 拦截 → UAC → 样本提交

  步骤6 — 关闭任务栏资讯（需手动操作）
    弹出操作教程窗口，请根据提示手动关闭后点击"确定"继续

  步骤7 — 其他系统优化
    显示文件扩展名 + 隐藏文件 + 桌面显示"此电脑" + 关闭开始菜单建议

  步骤8 — USB 驱动安装
    自动安装 software\usb_driver\android_winusb.inf

  步骤9 — MCR3512 控制键安装
    自动查找诺咪雅MCR3512文件夹，静默安装 exe 并导入 reg

  步骤10 — Windows 激活检测
    检测系统是否已激活 → 未激活则尝试内置 KMS 激活
    → 失败则自动查找并运行 HEU_KMS_Activator
    → 若激活工具被删除则从 jihuo.zip 解压后运行
    （运行激活工具前会暂停 Defender 实时防护，完成后恢复）

4.2  完整安装（设置 + 软件，约 10-30 分钟）
──────────────────────────────────
  1. 首先弹出软件勾选窗口，默认全选，可自由勾选/取消
  2. 点击"确认"后，先执行全部系统设置（同 4.1）
  3. 然后依次安装勾选的软件：
     - 每种安装包依次尝试多种静默参数（/S → /VERYSILENT → /quiet）
     - 若静默安装未成功，弹出选择窗口：
         [尝试下一种方法] — 换一种静默参数继续
         [跳过此软件]     — 不装这个，继续下一个
         [停止所有安装]   — 终止后续所有软件安装
     - 支持按键操作：1=换方法 / 2=跳过 / 3=停止
  4. burnaware 刻录软件安装后自动将桌面快捷方式改为"案件数据专用刻录软件"
  5. 完成后显示安装统计：总数 / 已跳过 / 新装 / 失败

4.3  还原设置（约 1-2 分钟）
──────────────────────────────────
  逐项恢复系统设置为 Windows 默认：
    壁纸 → 默认 / 锁屏 → 清除 / 更新服务 → 启用
    电源管理 → 默认 / 防火墙 → 启用 / Defender → 启用
    SmartScreen → 启用 / UAC → 启用 / 任务栏资讯 → 恢复
  注意：火绒安全不会被卸载，已安装软件不会被卸载
  完成后弹出软件卸载页面，可按需勾选卸载

4.4  映射网络驱动器
──────────────────────────────────
  自动断开已有 Z: ~ T: 映射，然后逐个映射：
    Z: → \\10.10.10.12\d
    Y: → \\10.10.10.13\d
    X: → \\10.10.10.14\d
    W: → \\10.10.10.15\d
    V: → \\10.10.10.16\d
    U: → \\10.10.10.17\d
    T: → \\10.10.10.18\d
  完成后弹窗汇总成功/失败数量

============================================
【五、软件文件夹准备规范】
============================================

将需要安装的所有软件放入 software/ 文件夹：

  支持格式：.exe  /  .msi
  支持目录：MCR3512 控制键等可放在子文件夹内（脚本会自动搜索）

  常见软件清单（文件名仅供参考）：
    火绒安全         sysdiag-full-*.exe
    谷歌浏览器        chrome_installer*.exe
    WPS Office        *WPS*.exe
    微软运行库合集    微软常用运行库合集.exe
    光盘刻录          光盘刻录burnaware*.exe
    iTunes            *iTunes*.msi

  注意事项：
  - 文件名不要包含特殊字符（可能导致路径解析错误）
  - 火绒安装包将在软件勾选页面中出现，用户可自行选择是否安装
  - 激活工具 HEU_KMS_Activator*.exe 不会被当作普通软件安装

============================================
【六、壁纸和锁屏】
============================================

将壁纸和锁屏图片放在脚本根目录（不是 software 文件夹）：

  壁纸：wallpaper.jpg  / wallpaper.png  / wallpaper.bmp
  锁屏：lockscreen.jpg / lockscreen.png / lockscreen.bmp

不提供对应文件则自动跳过该步骤。
锁屏设置仅支持 Win10/11 专业版和企业版，家庭版可能无效。

============================================
【七、Windows 激活说明】
============================================

自动激活流程：
1. 检测系统是否已激活
2. 未激活 → 尝试内置 slmgr KMS 激活
3. 失败 → 查找 software/ 下 HEU_KMS_Activator*.exe
4. 未找到 → 查找 software/jihuo.zip 并解压到临时目录
5. 运行激活工具（先 /smart 智能激活，失败则 /kms38）

防御杀软误删：
- 运行激活工具前会自动暂停 Defender 实时防护
- 建议同时将 jihuo.zip 放在 software/ 作为备份
- 若 zip 也被删除，请重新放入后重试

============================================
【八、日志文件】
============================================

每次运行自动在 logs/ 文件夹生成日志文件：

  setup_时间戳.log     — 系统设置日志
  fullsetup_时间戳.log — 完整安装日志

============================================
【九、常见问题】
============================================

Q: 运行后没有反应？
A: 可能需要管理员权限，右键 .bat 文件 → 以管理员身份运行。

Q: GUI 界面文字显示乱码？
A: 确认系统区域设置为中文（中国），.bat 文件已包含 chcp 65001。

Q: 壁纸/锁屏没有生效？
A: Win10/11 家庭版不支持组策略锁屏，壁纸需要重启资源管理器或注销后生效。

Q: 软件安装卡住了？
A: 可能是安装包弹出不可见的对话框。尝试点击任务栏上的安装程序窗口，
   或按 Alt+Tab 切换到安装向导窗口。

Q: 激活工具被删除了？
A: 将被删除的 jihuo.zip 重新放入 software/ 文件夹，重新运行脚本。
   脚本会自动暂停 Defender 防护后再解压和运行激活工具。

Q: 任务栏资讯关不掉？
A: 本工具会在执行到该步骤时弹出操作教程，请按教程手动操作，
   操作完成后点"确定"继续。

Q: 某些 Defeder 设置没有生效？
A: Windows 10/11 的篡改防护可能会阻止部分注册表修改，属正常现象。
   脚本已尽可能处理，剩余项不影响日常使用。

============================================
【十、注意事项】
============================================

1. 必须以管理员身份运行（各类 .bat 启动脚本会自动申请）
2. 部分设置需要重启电脑后才能完全生效
3. 运行过程中请勿关闭黑窗（PowerShell 控制台窗口）
4. 每次运行都会在 logs 文件夹生成新日志，旧日志不会被覆盖
5. 还原设置不会卸载火绒和已安装的软件
6. 软件安装时若弹出安装向导窗口，按提示手动完成即可
7. 若需完全恢复出厂状态，建议直接重装系统
8. 修改 .ps1 脚本后，需确保保存为 UTF-8 BOM 编码
